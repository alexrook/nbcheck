Статус DNS и NetBIOS

Справочная информация

Что такое NetBIOS?
Разрешение имени NetBIOS — это процесс определения IP-адреса по имени NetBIOS. Имя NetBIOS представляет собой 16-байтовый адрес, используемый для идентификации в сети ресурса NetBIOS. Имя NetBIOS может быть либо уникальным (эксклюзивным), либо групповым (неэксклюзивным). Когда процесс NetBIOS соединяется с конкретным процессом на конкретном компьютере, используется уникальное имя. Когда процесс NetBIOS соединяется с несколькими процессами на нескольких компьютерах, применяется групповое имя.
(c) Microsoft
Что такое DNS?
DNS (англ. Domain Name System — система доменных имён) — компьютерная распределённая система для получения информации о доменах. Чаще всего используется для получения IP-адреса по имени хоста (компьютера или устройства), получения информации о маршрутизации почты, обслуживающих узлах для протоколов в домене.
(c) Википедиа

В процессе накопления статистики использования Интернет через Squid (see Squid stats) было обнаруженно, что для некоторых ip-адресов DNS-имена не совпадают с NetBIOS-именами. Что может вызвать проблему для клиентского ПО, с порядком разрешения NetBIOS-имен, к примеру, HOSTS, DNS, cache, WINS, BCAST (see Samba Name Resolution)

Причиной может являтся переименование клиентской машины и не внесение соответствующих правок в DNS. А также отсутствие в сети развернутого сервера WINS.

Настоящее ПО, опираясь на собранную базу клиентских ip-адресов, периодически запрашивает сеть по методу BCAST (широковещательный запрос) разрешая ip-адреса в NetBIOS-имена. Метод BCAST работает для включенных в момент запроса Windows машин получая от них NetBIOS-имя указанное в System Properties ( к примеру )

Запросы выполняются дважды в день в рабочее время для ip-адресов, у которых, приложением, еще на найдено NetBIOS-имя. И, еженедельно в рабочее время, для всей базы ip-адресов, для синхронизации с сетью.

Вниманию администратора предлагается колонка NetBIOS-names. Для ip-адреса, ни одно из NetBIOS-имен которого, не совпадает с DNS-именем (игнорируя регистр букв и домен машины) рекомендуется изменить имя клиентской машины, или внести правки в DNS.
